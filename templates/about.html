About Project

This is our project for the VCU Capstone. We have created a security engine using Python and its data science packages, Anacondas, Numpy and Pandas in conjunction with Flask for hosting it via a web server. The user will enter a log file and the program will data mine it looking for various pieces of information in order to find anomalies that could be potential threats to the network the engine is protecting.

Our project searches though authentication log and reports on typical signs of possible intrusion as well as supplies whatever CIRT team with analytics on the data collected. Our project deliverables include means of analysis on different anomalies including successive repeated failures, location based login anomalies, and excessive fails from one location. Other deliverables included output to a local host web interface for easy presentation of anomalies and statistics, and developing means to easily search logs based on username for in depth study by the security team.
