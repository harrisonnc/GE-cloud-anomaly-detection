About Project

This is our project for the VCU Capstone. We have created a security engine using Python and its data science packages, Anacondas, Numpy and Pandas in conjunction with Flask for hosting it via a web server. The user will enter a log file and the program will data mine it looking for various pieces of information in order to find anomalies that could be potential threats to the network the engine is protecting.
